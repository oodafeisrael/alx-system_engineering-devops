Web infrastructure design: DNS- A system to translate domain names into IP addresses. web server: a software that serves web pages to clients upon their request, it does this over the protocol HTTP. database: Is a collection of information that is stored and organized so that it can be easily accessed, updated and managed. A server is a device, a virtual device or computer program or providing functionality for other programs or devices, called “clients”. HTTPS- A version of HTTP that secure the traffic between your browser and the website by encrypting it.  a codebase: Is the collection of source code that is used to build a software system. TCP/IP: Transmission Control Protocol/Internet Protocol, is a suite of communications protocols used to interconnect network devices on the Internet or any private network.
